Parsing_selenium
Описание
Данный проект демонстрирует примеры использования библиотеки Selenium для парсинга веб-сайтов. В качестве конкретного примера парсинга будет использоваться веб-ресурс https://bus.gov.ru/registry для извлечения определенных данных.

Файлы
1. parsing_selenium.ipynb - Файл Jupyter Notebook, содержащий примеры использования Selenium для парсинга веб-сайтов, с акцентом на веб-ресурсе https://bus.gov.ru/registry.
2. README.md - В данном файле содержится описание проекта, инструкции по запуску и другая полезная информация.
3. requirements.txt - Файл с зависимостями проекта, необходимыми для установки перед запуском проекта.

Использование
Для запуска проекта необходимо установить Python и все зависимости из файла requirements.txt. После установки библиотек и chromedriver, вы можете запустить parsing_selenium.ipynb и следовать инструкциям внутри файла для примеров использования Selenium на веб-ресурсе https://bus.gov.ru/registry.

Требования
Для корректной работы проекта необходимо убедиться, что у вас установлен Python и все зависимости из requirements.txt. Также убедитесь, что у вас имеется chromedriver.exe и он доступен для использования.
